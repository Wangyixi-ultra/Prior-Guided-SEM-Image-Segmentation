# Prior-Guided SEM Image Segmentation

This repository contains the implementation code for prior-guided SEM image segmentation.

## Repository Structure

- `U-Mamba/`: U-Mamba based segmentation framework
  - `umamba/`: Core implementation
  - `configs/`: Configuration files
  - `evaluation/`: Evaluation scripts
  - `assets/`: Assets and visualizations
- Root `.py` scripts: Data processing, model training, inference and evaluation scripts
